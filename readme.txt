# RTES-Challenge
 Final Project for NYU Tandon EC-GY-6483


### How to Use:

- Compile and upload code to the board.
- add a key by pressing user button(blue) present 
- Record the gesture 
- to unlock replicate the recorded gesture 
- when unlocked the screen will be in green colour with a tick symbol on it and the green colour LED lights up
= when the recorded pattern does match the key pattern the screen wont be unlocked a screen with red background appears asking us to try again and a red LED glows up 
- the black button on the board is for reset and the blue one is for input
